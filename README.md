## Structure
```
  /configure
  |- default.ts
  /src
  |- /controllers
  |- /middlewares
  |- /models
  |- /schema
  |- /services
  |- /utils
  |- app.ts
  |- routes.ts
```

## Features
* Allows to create an user account
* Allows to create a session with your user account
* Allows to create a product 
* Allows to update a product
* Allows to delete a product
* You can log out from the session
